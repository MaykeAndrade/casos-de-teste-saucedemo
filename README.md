# 🧪 Casos de Teste - SauceDemo

Projeto desenvolvido com o objetivo de demonstrar a criação e organização de casos de teste, simulando atividades reais de um profissional de Quality Assurance (QA).

---

## 🧠 Objetivo

Este projeto tem como finalidade:

* Estruturar casos de teste de forma clara e objetiva
* Cobrir cenários principais de uma aplicação web
* Demonstrar pensamento analítico e organização de testes

---

## 🌐 Sistema testado

Aplicação utilizada:

https://www.saucedemo.com/

## 🧪 Escopo dos testes

Os testes foram elaborados com foco nos principais fluxos da aplicação:

* Login
* Listagem de produtos
* Carrinho de compras
* Remoção de itens
* Checkout

---

## 📄 Casos de teste

Os casos de teste estão documentados no arquivo:

👉 `test-cases.md`

Incluindo:

* Cenários positivos e negativos
* Pré-condições
* Passos detalhados
* Resultados esperados
* Prioridade
* Tipo de teste
